# Korbz Solutions

Premium event curation & coordination — production website.

🌐 **Live:** https://korbz-glamour-glow-launch-main.vercel.app

## Stack
- React 18 + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- React Three Fiber (3D hero scene)
- Deployed on Vercel

## Local development
```bash
npm install --legacy-peer-deps
npm run dev
```
